Como usar em qualquer dispositivo

Para acessar de celular e computador, você precisa configurar seu próprio Firebase gratuito. É simples:

1. Acesse console.firebase.google.com
2. Crie um projeto gratuito
3. Clique em "Firestore Database" → criar banco
4. Em "Configurações do projeto" copie os dados do SDK
5. Abra este arquivo HTML em um editor de texto e substitua os campos COLE_AQUI pelos seus dados
6. Hospede o arquivo no GitHub Pages (gratuito) para acessar de qualquer lugar

Até configurar o Firebase, tudo salva no navegador atual automaticamente.
